In Pandas, method chaining enables us toÂ perform operations on a DataFrame without breaking up each operation into a separate line or creating multiple temporary variables.

Solutions

  • import pandas as pd


def findHeavyAnimals(animals: pd.DataFrame) -> pd.DataFrame:
    return animals[animals['weight'] > 100].sort_values('weight', ascending=False)[
        ['name']
    ]
